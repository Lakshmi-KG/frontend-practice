Border Styles in CSS

The border-style property defines the type of border around an element. You can use it to make borders solid, dashed, dotted, or invisible.

Style-width

Sets the thickness of the border.

Broder-color

Sets the color of the border.

Broder-radius

Rounds the corners of the border.

Types of Border Styles:

1.none --no broder(defalut is none)
2.solid -- a single solid line
3.dashed --A series of short dashes
4.dotted --A series of dots
5.double --Two solid lines with space between them
6.groove --Appears as a carved 3D groove
7.ridge --Appears as a raised 3D border
8.inset --Appears as if the element is embedded
9.outset --Appears as if the element is coming out
